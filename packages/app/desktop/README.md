# Desktop 应用

桌面应用版本的 AI Agent 应用。

## 概述

Desktop 应用基于 agent-core 框架构建，提供跨平台桌面环境下的 AI 交互体验。

## 功能特点

- 🖥️ 图形用户界面，用户体验友好
- 🪟 跨平台支持（Windows/macOS/Linux）
- 🔔 系统通知集成
- 📁 本地文件系统访问
- 🔌 可离线运行

## 目录结构

```
desktop/
├── src/          # 源代码
├── assets/       # 静态资源
├── src-tauri/    # Tauri 后端（如果使用 Tauri）
├── electron/     # Electron 主进程（如果使用 Electron）
└── README.md     # 本文件
```
