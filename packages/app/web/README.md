# Web 应用

Web 浏览器版本的 AI Agent 应用。

## 概述

Web 应用基于 agent-core 框架构建，提供浏览器环境下的 AI 交互体验。

## 功能特点

- 🌐 无需安装，浏览器即可访问
- 📱 响应式设计，支持移动端
- ☁️ 云端部署，随时可用
- 🔗 易于分享和协作
- 🔄 自动更新

## 目录结构

```
web/
├── src/          # 源代码
├── public/       # 静态资源
├── dist/         # 构建输出
├── index.html    # 入口 HTML
└── README.md     # 本文件
```
